・汎用PCとサーバーの違いについて
→ハードウェア構成としては基本的に変わらない
→サーバーに求められる要素として、耐久性、安定稼働、セキュリティー対策などが挙げられる

・オンプレ・クラウドとは、その違いについて
オンプレ：システム運用上で必要なソフトウェアやハードウェアを自社で保有・管理して構築する運用形態
クラウド：クラウドベンダーが用意したサーバーなどのリソースの利用量に応じて料金を支払う運用方式

・AWS・GCP・Azure(3大クラウド)の違いについて
AWS：Amazonによって提供されるクラウドサービス(グローバル展開に強い)
GCP：Googleによって提供されるクラウドサービス(サービスが豊富)
Azure：Microsoftによって提供されるクラウドサービス(同社サービスとの親和性が高い)

・OSについて
→Operating Systemの略であり、PC操作やアプリ使用の土台となる基本ソフトウェアを指す
→ソフトウェアの種類の一つで、システム全体を管理するソフトウェアとなる

・Windows・Linuxのそれぞれの特徴について
Windows：Microsoftが開発したOSで、基本的操作はGUIとなるため直感的な操作が可能
Linux：主にITサービスのサーバーを稼働させるOSとして利用される

・Amazon Linux 2について
→AWSのLinuxOS
→AWSの多くのサービスと容易に統合できる点が最大の特徴

・AWS Cloud9 について
→AWSのIDE(統合開発環境)
→構築がクラウド環境のみで環境開発できる

・GUI・CUIとその違いについて
GUI：Graphical User Interfaceの略
→画像やアイコンをマウスカーソルといった直感的な操作でコンピュータを操作する
CUI：Character User Interfaceの略
→文字と記号によってコンピュータを操作する

・Linuxコマンドについて
→LinuxOSのCUI環境において作業・処理する場合の操作命令のこと

・Linuxコマンド例について
pwd(print working directory)：現在のディレクトリパスを調べる
mkdir(make directory):ディレクトリを作成する
cd(change directory)：別のディレクトリに移動する
ls(list segments)：ディレクトリ配下を確認する
touch：ファイルを作成する
→mkfile：ファイルサイズを指定してファイル作成
cp(copy)：ファイルやディレクトリをコピーする
rm(remove)：ファイルやディレクトリを削除する
sudo(substitute user do)：スーパーユーザー権限でコマンドを実行する
→su(substitute user)：ユーザーの切り替え
